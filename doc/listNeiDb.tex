\section*{Introduction}
A lot of \ty{neighbors'} programs allow remote execution. These programs
use databases provided by the \ty{neighbors} server \ty{never}. Every program
that provides an option for remote execution also provides an option, for 
specifying a certain database. In order to name a specific database, a user
has to be aware of all available databases. This program does exactly that.
It queries \ty{never} and prints out all available databases.
