\section*{Introduction}
Several Neighbors programs are based on the Neighbors database, \ty{neidb}.
These programs can work with either a local copy of \ty{neidb} or a remote 
copy. Users can specify the remote \ty{neidb} by choosing among those hosted
on the Neighbors server, \ty{never}. The program \ty{listNeiDbs} lists the
databases available.
