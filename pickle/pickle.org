  \section*{Introduction}
  The program \ty{pickle} picks clades from trees. It reads a Newick
  tree with labeled clades. It then picks the clades requested by the
  user and prints their leaf labels. Alternatively, it prints the
  requested clade as a Newick tree, which can be rendered with a
  suitable tree drawing program. For example, we can use \ty{plotTree}
  to draw clade 9 of the test tree to get Figure~\ref{fig:tree}A.
  \begin{verbatim}
  $ pickle -t 9 pickle/test.nwk | plotTree
  \end{verbatim}

  \begin{figure}
    \begin{center}
      \begin{tabular}{ccc}
	\textbf{A} & \textbf{B} & \textbf{C}\\
	\includegraphics{../pickle/tree1.ps} & %% 4cm x 4cm
	\includegraphics{../pickle/tree2.ps} & %% 4cm x 4cm
	\includegraphics{../pickle/tree3.ps}   %% 4cm x 4cm
      \end{tabular}
    \end{center}
    \caption{Full tree for testing the program \ty{pickle} (\textbf{A}),
      the same tree with clade 8 removed (\textbf{B}), or with clade 8
      collapsed (\textbf{C}).}\label{fig:tree}
  \end{figure}

  The program \ty{pickle} also has a ``complement'' option to get
  everything \emph{but} the nodes in the target clade. For example, we
  can remove clade 8 from the tree in Figure~\ref{fig:tree}A to get
  Figure~\ref{fig:tree}B.
  \begin{verbatim}
  $ pickle -t -c 8 pickle/test.nwk | plotTree
  \end{verbatim}

  Instead of removing a node, we can collapse and replaced it by the
  number of leaves in its subtree. For example, we can collapse clade 8
  from the tree in Figure~\ref{fig:tree}A, to get Figure~\ref{fig:tree}C
  with $n=5$ where clade 8 used to be. Collapsing a node implies tree
  mode, \ty{-t}.
  \begin{verbatim}
  $ pickle -C 8 pickle/test.nwk | plotTree
  \end{verbatim}

  func tree2slice(v *nwk.Node, ns []*nwk.Node) []*nwk.Node {
	  if v == nil { return ns }
	  ns = append(ns, v)
	  ns = tree2slice(v.Child, ns)
	  ns = tree2slice(v.Sib, ns)
	  return ns
  }

	  if v == nil {
		  return n
	  }
	  if v.Child == nil {
		  n++
	  }
	  n = size(v.Child, n)
	  n = size(v.Sib, n)
	  return n
  }
